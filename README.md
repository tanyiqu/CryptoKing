# CryptoKing
CTF密码学常见加密解密工具



## 安装使用

```shell
# 需要安装PyQt5（使用与我相同版本的库）
pip install PyQt5==5.15.4 PyQt5-Qt5==5.15.2 PyQt5-sip==12.9.0

# 国内清华源
pip install PyQt5==5.15.4 PyQt5-Qt5==5.15.2 PyQt5-sip==12.9.0 -i https://pypi.tuna.tsinghua.edu.cn/simple
```
